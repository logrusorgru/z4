z4-dark
=======

Z4 dark theme for Xfwm4. The theme based on
[default XFCE theme](https://github.com/xfce-mirror/xfwm4/tree/1d0a4a95b47d6f2b524ec61bfb0ac48bb838911c/themes/default).
